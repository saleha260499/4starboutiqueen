# User Stories

## Epic 1: Customer Account

### US-001
As a new customer, I want to register an account so that I can purchase products.

### US-002
As a customer, I want to log in securely so that I can access my account.

### US-003
As a customer, I want to update my profile so that my information stays current.

### US-004
As a customer, I want to reset my password so that I can regain access if I forget it.

---

## Epic 2: Product Catalog

### US-005
As a customer, I want to browse products by category so that I can easily find what I need.

### US-006
As a customer, I want to search products by name so that I can quickly find a specific product.

### US-007
As a customer, I want to filter products by price, size, and colour so that I can narrow my choices.

### US-008
As a customer, I want to view detailed product information so that I can make an informed purchase.

### US-009
As a customer, I want to see trending and best-selling products so that I can discover popular items.

---

## Epic 3: Shopping Cart & Wishlist

### US-010
As a customer, I want to add products to my cart so that I can purchase multiple items.

### US-011
As a customer, I want to update quantities in my cart so that I can change my order before checkout.

### US-012
As a customer, I want to save products to my wishlist so that I can buy them later.

---

## Epic 4: Checkout & Orders

### US-013
As a customer, I want to place an order securely so that I can buy products online.

### US-014
As a customer, I want to view my order history so that I can track previous purchases.

### US-015
As a customer, I want to track my order status so that I know when it will arrive.

---

## Epic 5: Admin Panel

### US-016
As an administrator, I want to manage products so that the catalogue stays up to date.

### US-017
As an administrator, I want to manage customer orders so that I can process and fulfil them.

### US-018
As an administrator, I want to manage users so that customer accounts remain secure.

### US-019
As an administrator, I want to view sales reports so that I can monitor business performance.

---

## Epic 6: Social Impact

### US-020
As a customer, I want to learn about the company's charity initiatives so that I understand how my purchase contributes to social impact.

### US-021
As an administrator, I want to record monthly charity contributions so that the business maintains transparency.

---

## Future Epic: AI Features

### US-022
As a customer, I want to virtually try on outfits using AI so that I can visualise how they look before purchasing.

### US-023
As a customer, I want personalised outfit recommendations so that I can discover styles that suit my preferences.

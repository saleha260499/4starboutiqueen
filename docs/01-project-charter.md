# 1. Project Overview

4 Star Boutiqueen is a premium online fashion boutique specializing in women's clothing. The business currently sells products through WhatsApp and Instagram, where product updates, customer communication, and order processing are handled manually.

The purpose of this project is to build a modern, secure, and scalable e-commerce platform that improves the customer shopping experience, automates business operations, and serves as a production-quality portfolio project following professional Software Development Life Cycle (SDLC) practices.
